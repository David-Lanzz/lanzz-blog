## 💻 Getting Started with Lanzz Blog <a name="getting-started"></a>
> To get a local copy up and running, follow these steps.

### Prerequisites
In order to run this project you need:
1. Ruby installed on your machine
2. Text editor (VSCode, Sublime, Atom, etc.)
3. Ruby on rails installed on your machine

### Setup

Clone this [repository](https://github.com/David-Lanzz/lanzz-blog.git) to your desired folder:

```sh
  cd lanzz-blog
  git clone https://github.com/David-Lanzz/lanzz-blog.git
```

### Install
Install this project with:
```sh
  cd lanzz-blog
```

#
### Run tests
Run tests with:
```sh
stylelint
  rubocop
```
